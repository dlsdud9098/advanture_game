## 🔍 발생한 문제 & 해결 방법
- 아이템을 저장할 때

```json
[
    {
        ...
    },
    {
        ...
    }
]
```

이런 형식으로 저장되어 데이터가 많을 때 하나씩 검색해야 했다.  
이런 구조를 아래와 같이 변경했다.  

```json
{
    a: {
        name: a,
        discription: ...,
    },
    b: {
        name: b,
        discription: ...,
    }
}
```
이렇게 되면 단순히 key값을 입력하는 것만으로 데이터를 찾을 수 있다!
